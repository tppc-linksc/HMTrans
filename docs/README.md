# HMTrans 文档索引

本目录是 HMTrans / HM互传的产品、技术和开发基线。当前公开版本为 `v0.1`，仓库代码为 `v0.2.0` 发布候选；构建和代码检查通过不等于正式渠道、超大文件、后台锁屏和三台真机验收通过。

v0.2 的设计与已实现代码以 00～04 为基线，实际发布结论以验收清单和发布前检查报告为准。没有真机或正式包证据的项目必须明确标记“待实测”。

## 文档维护规则

- 产品范围变化时，先更新项目说明和产品交互方案。
- 架构、存储、后台或性能策略变化时，更新技术方案。
- 双端字段、消息顺序或错误码变化时，必须先更新传输协议，再改代码。
- 新增任务必须进入开发计划，并在验收清单中有对应验收项。
- README 和双端子项目 README 只保留摘要，详细定义以本目录为准。
- v0.1 已完成功能不再放入 v0.2 开发计划；只在“当前基线”和兼容章节保留必要说明。

## v0.2 可交互 UI 原型

### 第一版（基础交互）
- [Mac 端原型](design/v0.2-prototype/mac.html)：设备、任务、历史、设置、状态栏悬浮与拖拽发送模拟。
- [Pad 端原型](design/v0.2-prototype/pad.html)：设备、任务、历史、设置、后台传输和断点恢复模拟。
- 两个页面共享 `prototype.css` 与 `prototype.js`，所有可见按钮均有点击状态或反馈，适合直接评审和修稿。

### 第五版（完全客观事实驱动 · 全新视觉「Handprint」）
- [入口页](design/v0.2-prototype-v5/index.html)：含设计来源明文、Mac / Pad 一键切换、浅色 / 深色。
- [Mac 端原型（v5）](design/v0.2-prototype-v5/mac.html)：工作台三栏（左 = 当前这摞文件 / 中 = 设备单行表 / 右 = 这一轮的结果）；菜单栏 = 合并的工作台面板（不区分悬浮和拖拽两套），300ms 悬浮、真实 HTML5 拖放都进同一面板。
- [Pad 端原型（v5）](design/v0.2-prototype-v5/pad.html)：全屏工作台三栏 + 系统通知镜像（活动传输时显示在状态栏下沿，含暂停 / 取消）。
- **设计来源（明文，刻意脱离之前版本与项目 docs 描述）**：macOS 菜单栏原生能力 / macOS Finder 拖入文件型 drop 语义 / HarmonyOS Pad ArkUI 全屏 + 长时任务通知 / 人类心智「拖文件给隔壁 = 物理放一摞纸」/ 同类竞品 AirDrop 与华为分享。
- **形态决策**：主体 = 当前这摞文件（drop 区即主舞台），应用退到工具栏；设备 = 单行表不分块；状态 = 形状（●/◌/□）+ 文字 + 顺序，不只靠颜色；菜单栏悬浮和拖拽合并为同一工作台面板。
- **弃用清单**：渐变、玻璃、卡片四色状态条、侧边栏 tab、工程图装饰线、紫青配色、衬线工程图（v1–v4 任何版本的视觉语言）。
- 共享 `lab.css` / `app.js`。v2 / v3 / v4 三个目录已删除，旧版登记表已从本 README 撤除，仅保留 v1（`v0.2-prototype`）与 v5 两版可供对比。

### 第六版（Apple HIG 极简美学 · 仅基于 docs/00-05）
- [入口页](design/v0.2-prototype-v6/index.html)
- [Mac 端原型（v6）](design/v0.2-prototype-v6/mac.html)：macOS 原生窗口（交通灯 + 半透明侧边栏 + SF Symbols 导航）+ 顶部菜单栏模拟（300ms 悬浮弹出精简任务面板 / 真实 HTML5 拖放弹出发送面板）。
- [Pad 端原型（v6）](design/v0.2-prototype-v6/pad.html)：iPad 风格全屏（大标题导航 + 底部毛玻璃 Tab Bar）+ 活动传输横幅（系统通知镜像，含暂停 / 取消）。
- 视觉：Apple 系统色（#007AFF / #34C759 / #FF9500 / #FF3B30）、SF Pro 字体栈、inset grouped 列表、iOS 开关、分段控件、步进器、连续圆角、0.5px 分隔线、spring 动画。无渐变、无实验性装饰。
- 功能：四区导航 · 设备三态（绿/蓝/灰 + 文字标签）· 11 状态任务机 · 六位码配对 + 180s 倒计时 + 错码拒绝 · 接收策略三档 · 群组全部暂停/继续/取消/只重试失败 · 历史三类清除 · 端口分别修改 · 临时空间清理 · 诊断导出 · 隐私首次启动门禁 · 深浅色 · 键盘可达 · 状态不只靠颜色。

## 不在 v0.2 范围内

- 云端中转、云同步和公网传输。
- 用户账号、登录注册和联系人系统。
- 将文件、传输历史或配对关系上传到服务器。
- 为了兼容旧系统而牺牲 HarmonyOS 6.1 / API 23+ 和 macOS 26+ 的原生体验。
