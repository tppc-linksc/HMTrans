# HMTrans 公开文档索引

本目录只保存可以随开源仓库发布的产品、技术、协议、开发和验收资料。当前公开渠道版本为 `v0.1`，仓库代码为 `v0.2.0` 发布候选；代码构建通过不等于应用市场干净安装、超大文件、后台锁屏或三设备真机验收通过。

## 维护规则

- 产品入口或操作逻辑变化时，同步更新 00、01、05 和双端 README。
- 网络字段或消息顺序变化时，先更新 03，再同步双端代码和测试。
- 没有正式包或真机证据的项目必须标注“待实测”，不能写成已通过。
- 本项目没有 Apple Developer Program 账号，Mac 包保持 ad-hoc 签名、未经 Apple notarization；文档只说明事实和用户打开方式，不伪造签名、公证或验证结果。

## v0.2 之外

- 云端中转、云同步、公网传输和账号系统。
- 上传文件、历史、诊断或配对关系。
- TLS/内容端到端加密；当前六位码用于局域网人工配对门禁，不等于内容加密。
