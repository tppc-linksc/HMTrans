# HMTrans 公开文档索引

本目录只保存可以随开源仓库发布的产品、技术、协议、开发和验收资料。当前稳定版本为 `v0.2.1`；`v0.3.0` 已完成双端开发构建，目标是把 MatePad 画面单向投放到 Mac，目前处于真机验收前状态。

官网页面位于本目录的 `index.html`、`connection-help.html`、`privacy.html` 和 `site-assets/`。官网代表已经发布的稳定版本，在 v0.3 完成真机验收和发布前继续展示 v0.2.1，不能因为开发文档建立就提前宣传投屏已经可用。

## 文档维护规则

- 已完成旧版本只保留在 [版本演进](01-版本演进.md)，不继续维护多套过时产品、技术和验收文档。
- 产品入口或操作逻辑变化时，同步更新 00、02、06、根 README 和双端 README。
- 投屏字段或消息顺序变化时，先更新 04，再同步双端模型、解析器和测试。
- 没有正式包或真机证据的项目必须标注“开发中”“计划”或“待实测”，不能写成已通过。
- v0.3 没有发布前，不修改官网为已支持投屏；发布后再同步官网、隐私政策和应用市场说明。
- 本项目没有 Apple Developer Program 账号，Mac 包使用 ad-hoc 签名且未经 Apple notarization；文档只说明事实，不伪造签名、公证或验证结果。

## 固定产品边界

- 默认局域网直连，不提供云端中转、账号、云同步或公网传输。
- 文件和画面不得上传；诊断、历史和配对关系只保存在本机。
- v0.3.0 只做画面投放，不含声音、反向控制、副屏和内置录制。
- 屏幕画面属于敏感内容，公开发布前必须完成身份验证、会话加密、隐私说明和真机安全验收。
