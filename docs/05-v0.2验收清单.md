# v0.2 验收清单

“构建通过”“模拟器/本机通过”“正式包真机通过”必须分别记录。下面未勾选项是发布前外部验证，不是可以用文档替代的代码任务。

## A. 构建与安装

- [x] Mac Release 构建和 Swift Tests 通过。
- [x] HarmonyOS HAP 构建通过。
- [ ] Mac 在另一台干净机器按未公证应用流程首次打开成功。
- [ ] HarmonyOS 应用市场正式包干净安装和升级安装成功。
- [ ] 市场包与调试包签名切换说明清晰，不尝试直接覆盖不同签名包。

## B. 首启、隐私和权限

- [ ] 白色无图标启动页后出现应用自有隐私弹窗。
- [ ] 不同意后网络/目录服务不启动并退出。
- [ ] 同意后出现系统下载目录授权；允许后可写 `Download/HMTrans`。
- [ ] 拒绝目录授权仍可发现和发送；设置“检查”可重新申请并有 Toast。
- [ ] 应用市场下载版本无需用户预先进入系统设置手工授权。

## C. 发现与配对

- [ ] 双端真实 SSID、设备名和系统版本正确。
- [ ] 新设备必须输入对方六位码；错码/过期码拒绝。
- [ ] 双向确认后两端同时绿色在线，退出一端后 12 秒左右离线。
- [ ] 已配对离线设备可重连；同一设备不会出现两张卡。
- [ ] 移除本机信任后二次确认生效，下次必须重新配对。
- [ ] v0.2.x 覆盖升级保留安装身份和信任，双端无需输入配对码即可自动重连。
- [ ] Pad 清除应用数据、Mac 清除 `~/Library/Application Support/HMTrans` 后身份变化，旧信任不能直接连接，必须重新配对。
- [ ] Mac 同时从两个路径启动 HMTrans 时只有一个实例运行网络服务，另一个给出明确提示并退出。
- [ ] v0.1/v0.2 明确提示同时升级，不错误宣称兼容。

## D. 文件与文件夹

- [ ] 单文件双向发送，名称、大小和 SHA-256 一致。
- [ ] 小文件在接收端瞬时落盘后，发送端能消费提前到达的最终回执并进入“已完成”，不会停留“准备中”或循环重连。
- [ ] 多文件选择不漏文件。
- [ ] 深层嵌套文件夹还原原目录结构，用户看不到 ZIP/分片。
- [ ] 路径穿越、绝对路径、盘符和符号链接 ZIP 被拒绝。
- [ ] 重名目标生成唯一名称，不覆盖已有文件。
- [ ] 5 GB、20 GB+ 和 ZIP64 大文件夹通过。

## E. 任务控制和恢复

- [ ] 发送/接收暂停、继续、取消保留、取消删除均符合状态。
- [ ] 断网、切后台、进程重开后同任务 ID 从确认偏移继续。
- [ ] 锁屏、Pad 分屏/浮窗、Mac 睡眠/唤醒后恢复。
- [ ] 清理临时空间不会删除活动任务正在使用的文件；忙时提示重试。
- [ ] 错误显示可读原因和建议，不出现 `{}`。

## F. 多设备

- [ ] 同一文件同时发送到两个直连设备，进度和结果独立。
- [ ] 给设备 A 发送文件 1 的同时，给设备 B 发送文件 2。
- [ ] 一台失败不影响另一台完成，失败子任务可单独重试。
- [ ] A-B、B-C 已连而 A-C 未连时，不发生未经用户操作的 B 中继。
- [ ] Mac 最高 3、Pad 最高 2 个网络会话并发，更多任务安全排队。

## G. UI 与可访问性

- [ ] Mac 左侧整行和历史五段全部区域可点击；浅色/深色均清晰。
- [ ] Pad Dock 确认为原生 `IMMERSIVE + EXQUISITE`，三项位置/图标正确。
- [ ] Pad 横竖屏、分屏和最小浮窗不溢出，历史五段可点击。
- [ ] Pad 左滑删除、长按移除设备、清空确认符合触摸操作。
- [ ] 状态不只靠颜色表达，文字和图标同时存在。
- [ ] VoiceOver/屏幕朗读可识别导航、设备状态、开关、配对码倒计时、任务控制和删除操作。

## H. 设置、诊断和数据

- [ ] 四个开关重启后保持，并真实控制接收/发现/自动接收/后台保护。
- [ ] UDP/TCP 非法值不保存，合法值同时保存并重启服务。
- [ ] 诊断复制有反馈，内容无文件字节、完整路径、IP、配对码和指纹原值。
- [ ] 历史只由用户逐条、按设备或全部清理；不自动丢失。
- [ ] 诊断数据库长期运行只保留最近 500 条。

## I. 能耗与稳定性

- [ ] 空闲 30 分钟无后台长时任务，记录 CPU/能量影响。
- [ ] 连接风暴和超长控制行不会造成 OOM。
- [x] 自动化验证半开 TCP 对端会在操作超时后退出，不永久阻塞 Mac 传输线程。
- [ ] 连续传输/取消/重试 100 次无句柄、计时器或数据库泄漏。
- [ ] 低磁盘、端口占用、权限收回、网络切换均可诊断并恢复。
