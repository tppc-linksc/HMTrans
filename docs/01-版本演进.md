# 版本演进与后续路线

本文是 HMTrans 唯一的历史版本总览。已经完成的旧版本不再保留独立产品、技术、协议和验收文档；需要理解当前实现时，以当前版本文档和代码为准。

## 版本规则

- `0.x.0` 引入一组新的主要能力或产品边界。
- `0.x.y` 用于兼容性、稳定性、性能和交互修复，可以在不改变主要协议语义的前提下迭代。
- Mac 与 HarmonyOS 对外版本尽量保持一致；平台内部构建号分别递增。
- 尚未完成真机验收的能力只写为“计划”或“开发中”，不能在 README、官网或应用市场描述成已发布。

## v0.1：建立双端直连

### 主要实现

- 建立 macOS 与 HarmonyOS MatePad 双端工程。
- 在同一 Wi-Fi 中通过 UDP 发现设备，并通过 TCP 直接传输文件。
- 支持 Mac 选择文件发送、Pad 接收文件和基础进度展示。
- 文件保存到用户可见下载目录，不经过云端。

### 版本优化重点

- 验证局域网原文件互传的最小闭环。
- 解决 HarmonyOS 应用市场安装包首次启动、下载目录授权和发现服务启动顺序问题。
- 明确产品只做本地传输，不增加账号和云端中转。

## v0.2 / v0.2.1：把文件互传做成可长期使用的工具

### 主要实现

- 引入六位配对码、随机安装身份、设备信任与实时解除配对。
- 引入本地数据库、私有持久 Staging、统一任务状态和进程重启恢复。
- 支持文件夹归档与安全还原、1 MiB 分块、SHA-256、暂停、继续、取消、重试和断点续传。
- 支持一对多发送，不同目标拥有独立进度、结果和恢复点。
- 将连接、活动任务和历史整理为“连接 / 历史 / 设置”三个一级入口。
- 增加后台传输保护、下载目录检查、临时空间清理、端口设置、诊断信息、深浅色和窗口形态适配。
- Mac 增加菜单栏快捷入口、单实例保护和 DMG 分发；Pad 使用原生沉浸光感 Dock。

### v0.2.1 重点修复

- 修复小文件完成回执可能丢失、Pad 长时间停留“准备中”的问题。
- 修复同一 Mac 中开发副本与下载副本同时运行造成的身份/端口冲突。
- 将配对身份固定为应用安装数据：覆盖升级保留信任，清除数据后重新配对。
- 删除任一端设备时向对端同步解除配对，双方立即回到未配对状态。
- 收口深色模式、历史筛选、Pad Dock、网络名称、设备名称和版本展示。

## v0.3.0：MatePad 单向投屏到 Mac

### 主要实现

在已配对的 Mac 与 MatePad 之间增加低延迟画面投放，满足应用演示和使用 Mac 系统录屏录制演示视频的需要。

- Pad 使用 HarmonyOS 原生屏幕采集与硬件 H.264 编码。
- 通过独立实时协议把视频配置、关键帧和普通帧发送到 Mac。
- Mac 使用 VideoToolbox 解码，并在独立窗口中按比例显示画面。
- 支持普通窗口、画中画、全屏、横竖屏切换、主动停止、超时、断线重连和请求关键帧。
- 投屏入口复用现有已配对设备，不新增主导航页；每次会话仍显示系统录屏授权。
- 文件传输与投屏协议隔离，投屏不创建文件历史或暂存视频文件。
- 同一 Mac 最多接收两台 Pad；双路会话限制为不高于 1080P 级 / 30P，并提供投屏网络测试辅助判断局域网余量。

### v0.3.0 明确不做

- 音频采集与播放。
- Mac 反向触控、键盘、鼠标和快捷键控制。
- 把 MatePad 作为 macOS 扩展显示器。
- HMTrans 内置录制、剪辑、云端转发或公网投屏。

## v0.3.x：围绕投屏持续优化

下面的顺序是路线建议，不是已经承诺的固定发布日期：

| 版本方向 | 主要优化 |
| --- | --- |
| `v0.3.1` | 增加 Mac 主动请求 Pad 投屏；增加经 Pad 单独授权的触控、键盘和鼠标反向操作；同时继续优化首帧、重连、功耗和稳定性 |
| `v0.3.2` | 在平台权限和隐私规则允许的前提下增加系统/麦克风声音传输与同步 |
| `v0.3.3` | 动态码率、丢帧策略、更多 MatePad 机型适配和多路投屏体验优化 |

每个后续版本都必须保持：默认本地直连、用户主动授权、清晰停止入口、不采集无关数据、无账号和无云端中转。

`v0.3.1` 的远程启动和反向操作是两个独立授权能力：允许 Mac 请求开始投屏，不代表自动获得控制权限；Pad 用户可只允许观看、随时撤销控制或停止整条投屏会话。

## 更后续版本

`v0.4` 及以后暂不绑定具体功能。只有当 `v0.3.x` 的画面、声音和控制链路完成真实设备验收后，才确定下一阶段。候选方向可以记录在 issue 中，但不能提前写入稳定版功能清单。
