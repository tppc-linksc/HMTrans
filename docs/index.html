为什么做它

只解决一件事：Mac 和 MatePad 稳定互传原文件

01

局域网直传

设备在同一 Wi-Fi 下自动发现，文件在两端之间直接传输，不经过第三方服务器。

02

保留原文件

面向图片、视频、文档和工程文件等场景，避免聊天工具压缩或改写文件。

03

双端原生

macOS 使用 SwiftUI / AppKit，HarmonyOS 使用 ArkTS / ArkUI，优先贴近系统体验。

04

传输可见

当前传输和历史记录展示方向、进度、速度、大小、格式和结果，失败原因可追踪。

05

可恢复与多设备

v0.2 发布候选支持暂停、继续、取消、断点恢复、文件夹还原和多个已连接设备独立任务。

06

本地隐私

无账号、无云端；应用自己的隐私确认完成后才启动局域网服务，发送内容只来自用户主动选择。

支持平台

为桌面和 MatePad 工作流设计

HM互传优先服务个人设备之间的高频文件流转：Mac 上拖拽或选择文件，MatePad 上接收、查看保存位置，再继续编辑或归档。

macOS

菜单栏常驻、主窗口传输列表、Dock 拖拽接收入口。

HarmonyOS

ArkUI 界面、系统文件选择、下载目录保存、平板横屏和窗口化优先。
