HM互传隐私政策

更新日期：2026 年 7 月 7 日

一、应用说明

HM互传是一个用于 macOS 与 HarmonyOS 设备之间进行局域网文件传输的工具。应用的核心能力是在同一局域网内发现设备并直接传输文件。

二、信息收集

HM互传不提供云端账号系统，不主动收集、上传或出售用户个人信息。文件传输发生在用户选择的设备之间，应用不会把文件上传到第三方服务器。

三、本地使用的信息

为完成设备发现、连接和传输记录展示，应用可能在本地读取或保存设备名称、局域网地址、传输文件名、文件大小、传输结果、保存位置和校验状态。这些信息用于展示当前传输和历史记录。

四、权限使用

应用可能请求网络访问、文件选择、文件保存、下载目录读写或通知等权限。相关权限仅用于设备发现、文件发送、文件接收、保存结果展示和传输状态提醒。

五、数据存储与删除

接收文件默认保存到用户可见目录。传输记录保存在本地，用于展示历史结果。后续版本会提供删除单条传输记录和清空记录能力；清空记录不等同于删除已保存文件，除非用户明确选择同时删除文件。

六、联系我们

如需反馈问题或提出隐私相关请求，可以通过 GitHub 仓库提交 Issue。
